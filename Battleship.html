Welcome to Battleship

This application is my recreation of the classic Battleship game. The player and the enemy take turns trying to destroy each others warship until one side has no ships remaining. Below you will find a short video demoing the application along with a button to view the source code.

View Source Code

Battleship is a Windows Forms application created using C#. It features a custom algorithm designed to allow the enemy ships to be placed randomly and dynamically on the map upon loading the game. The player also has the choice to use the algorithm to load their ships in or if they would like to be more strategic they can pick their ship locations. Battleship also features layered architecture which uses a custom made class library that holds vital information that drives a lot of the core game play.
